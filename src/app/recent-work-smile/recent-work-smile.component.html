Projet Skillquiz - SMILE

Retour aux Projets

📄Présentation

Stage réalisé chez SMILE, visant à concevoir une application web interne (Skillquiz) pour évaluer techniquement les candidats et les employés par le biais de quiz techniques lors des entretiens d'embauche.

🎯Contexte et Objectifs

Optimiser le processus de recrutement avec une application offrant des quiz adaptés et des statistiques pour évaluer les candidats et permettre aux spécialistes internes d'obtenir des badges de compétence.

  • Equipe : 2 stagiaires Java, Tech lead
  • Méthodologie : Agile, avec réunions quotidiennes

💻Technologies Utilisées

  • Frontend : Angular
  • Backend : Java Spring
  • Base de données : PostgreSQL dans Docker
  • CI/CD : GitLab, Docker, Harbor

🔧Détails Techniques

Refonte de l'API backend en architecture multicouche (controller, service, DAO), mise en place de la CI/CD et automatisation du déploiement avec Docker et GitLab CI/CD, enregistrement des images Docker dans Harbor.

📝Missions Principales

  • Refonte de l'API backend et création d'une architecture modulaire.
  • Mise en relation entre l'interface Angular et l'API pour la recherche des spécialistes.
  • Implémentation du passage de quiz avec génération, minuteur, et validation des réponses.
  • Configuration de la CI/CD pour un déploiement fiable via Docker et Harbor.

📈Compétences Acquises

Développement d'API avec Java Spring, intégration d'Angular, pratiques CI/CD, gestion d'architecture backend, amélioration de la collaboration en équipe et de la communication.
